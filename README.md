OneNote 전자필기장 스크롤 리모컨 (OneNote Scroll Remocon)

![alt text](https://img.shields.io/badge/GUI-PyQt6-green.svg)


![alt text](https://img.shields.io/badge/Automation-pywinauto-orange.svg)


![alt text](https://img.shields.io/badge/OS-Windows-blue)

OneNote 전자필기장 스크롤 리모컨은 사용자가 Windows 환경에서 OneNote 애플리케이션의 특정 섹션이나 페이지를 빠르고 정확하게 선택하고 화면 중앙으로 자동 정렬할 수 있도록 돕는 유틸리티입니다. 특히, 페이지 목록이 길 때 선택한 항목이 화면 밖으로 밀려나는 문제를 해결하여 사용 편의성을 극대화합니다.

주요 기능

자동/수동 OneNote 연결:

실행 중인 OneNote 창을 자동으로 감지하고, 이전에 연결했던 창을 기억하여 재시작 시 자동으로 연결을 시도합니다.

UWP(스토어) 및 클래식 버전 OneNote 모두 지원합니다.

선택 항목 중앙 정렬:

OneNote에서 현재 선택된 페이지 또는 섹션 항목을 자동으로 스크롤하여 화면의 중앙에 위치시킵니다.

이는 발표, 강의, 필기 중 빠르게 포커스를 전환할 때 유용합니다.

즐겨찾기 (Favorites) 관리:

자주 사용하는 섹션이나 페이지를 그룹화하여 계층적으로 관리할 수 있습니다.

더블클릭 한 번으로 해당 OneNote 창에 연결하고, 해당 섹션을 선택한 후, 자동으로 중앙에 정렬합니다.

섹션 연결 실패 시, 즐겨찾기 이름 앞에 (구) 접두사를 붙여 사용자에게 알립니다.

전자필기장 검색 및 이동:

텍스트 검색을 통해 특정 섹션을 찾아 자동으로 선택하고 중앙에 정렬하는 기능을 제공합니다.

데이터 관리:

즐겨찾기 목록을 JSON 파일로 내보내거나 가져올 수 있어, 설정 백업 및 공유가 용이합니다.

유연한 UI:

PyQt6 기반의 깔끔한 다크 테마 UI를 제공합니다.

설치 및 실행

이 애플리케이션은 Python 기반이며, Windows 자동화를 위해 pywinauto 라이브러리가 필요합니다.

1. 필수 환경 (Source Code 실행 시)

Python 3.8 이상 환경이 필요합니다.

code
Bash
download
content_copy
expand_less
# 필요한 라이브러리 설치
pip install pywinauto PyQt6
2. 실행 (Source Code)

다운로드 받은 경로에서 다음 명령을 실행합니다.

code

python OneNote_Scroll_Remocon.py

팁: PyInstaller 등으로 빌드된 실행 파일(EXE)을 사용하면 Python 환경 설정 없이 바로 실행할 수 있습니다.

사용 방법
1. OneNote 창 연결

애플리케이션을 실행하면 자동으로 이전 연결 정보를 기반으로 OneNote 창에 재연결을 시도합니다.

자동 연결: 상태 표시줄에 (자동 재연결) 메시지가 뜨면 성공입니다.

수동 연결:

중앙의 "OneNote 창 목록" 그룹에서 실행 중인 OneNote 목록을 확인합니다.

연결을 원하는 OneNote 창을 더블클릭합니다.

연결됨: '창 제목' 메시지를 확인하고, 아래 기능들을 활성화합니다.

다른 앱에 연결...: OneNote가 아닌 다른 일반적인 창(예: 브라우저, 다른 메모 앱)에 연결하여 기능을 테스트하거나 사용할 수 있습니다.

2. 현재 선택된 항목 중앙 정렬

연결이 완료된 후, OneNote 창에서 페이지나 섹션을 마우스로 클릭하여 선택합니다.

현재 선택된 항목 중앙으로 정렬 버튼을 클릭합니다.

OneNote의 페이지 목록 또는 섹션 목록에서 선택된 항목이 화면 중앙에 보이도록 자동으로 스크롤됩니다.

3. 전자필기장 검색

연결된 OneNote 창 내에서 특정 섹션을 찾고 싶을 때 사용합니다.

전자필기장 검색 입력창에 원하는 섹션의 정확한 이름을 입력합니다. (페이지 이름 아님, 섹션 이름)

전자필기장 위치 버튼을 클릭하거나, 입력창에서 Enter를 누릅니다.

앱이 해당 섹션을 찾아 선택하고, 자동으로 중앙 정렬을 시도합니다.

4. 즐겨찾기 (Favorites) 관리 및 사용

즐겨찾기 탭을 사용하여 자주 이동하는 섹션을 등록합니다.

버튼	기능	설명
현재 섹션 추가	현재 OneNote에 선택된 섹션을 즐겨찾기에 등록합니다.	등록 시 표시될 이름을 지정할 수 있습니다.
그룹 추가	즐겨찾기 항목들을 분류할 새 그룹을 만듭니다.	그룹 내에 섹션이나 다른 그룹을 포함할 수 있습니다.
이름 바꾸기 (F2)	선택된 그룹이나 섹션의 이름을 변경합니다.	
삭제 (Del)	선택된 항목을 삭제합니다. 그룹 삭제 시 하위 항목도 함께 삭제됩니다.	
▲ 위로 / ▼ 아래로	선택된 항목의 순서를 변경합니다.	

즐겨찾기 사용:

등록된 섹션 항목을 더블클릭하면, 해당 섹션이 등록된 OneNote 창이 활성화되고, 해당 섹션이 선택되며 중앙으로 정렬됩니다.

그룹 항목을 더블클릭하면, 그룹이 펼쳐지거나 접힙니다.

참고: 연결 실패 시 이름 자동 변경

즐겨찾기 항목을 활성화할 때 OneNote 내에서 해당 섹션을 찾는 데 실패할 경우, 해당 항목의 이름 앞에 (구) 접두사가 자동으로 붙여집니다. 이는 OneNote에서 해당 섹션이 삭제되었거나 이름이 바뀌었음을 나타내므로, 사용자는 이 항목을 수정하거나 삭제할지 결정할 수 있습니다.

5. 데이터 관리 (내보내기/가져오기)

메뉴바의 파일 항목을 통해 즐겨찾기 데이터를 관리할 수 있습니다.

즐겨찾기 내보내기...: 현재 설정된 즐겨찾기 목록을 JSON 파일로 저장합니다.

즐겨찾기 가져오기...: 저장된 JSON 파일을 불러와 현재 즐겨찾기 목록을 대체합니다.

기술 정보 및 문제 해결

자동화 백엔드: 이 앱은 pywinauto 라이브러리의 UIA (UI Automation) 백엔드를 사용하여 OneNote UI 요소를 제어합니다.

캐싱: OneNote의 페이지 목록(Tree Control)을 한 번 캐시하여 반복적인 스크롤 작업의 속도를 향상시킵니다.

중앙 정렬 알고리즘: 단순히 ScrollIntoView만 사용하는 것이 아니라, 항목의 위치를 확인한 후 작은 스크롤 단위로 목표 위치에 도달할 때까지 반복적으로 조정하여 정확한 중앙 정렬을 수행합니다.

알려진 제약 사항

OneNote의 페이지/섹션 목록이 비활성화되거나 완전히 다른 패널로 변경되는 경우, 연결이 일시적으로 끊길 수 있습니다. 이 경우, 새로고침 버튼을 눌러 목록을 다시 스캔하거나, 현재 선택된 항목 중앙으로 정렬을 시도하여 컨테이너를 재탐색하도록 유도해야 합니다.

pywinauto는 윈도우 기반 앱에만 적용 가능합니다. (macOS, Linux 환경은 지원하지 않습니다.)
